# トレーニング記録ダッシュボード

メモアプリに書いている筋トレ記録を、**強さ・量・バランス・継続・体**の観点で可視化する独立ツール。
人生管理（mylife）とは**完全に別物**です。

## 使い方

`index.html` をダブルクリックしてブラウザで開くだけ。データは `data.js` に入っています。
（グラフ描画に Chart.js と Google Fonts をネットから読み込むため、初回はネット接続が必要）

## 見られること（4つの問いに対応）

1. **強さ** — 種目ごとの**推定1RM の推移**（Epley式 `重量×(1+回数/30)`、1〜12repが対象）。
   生の最大重量は乱高下するが、推定1RMは「本当に強くなったか」を滑らかに示す。
2. **量とバランス** — 部位別ボリューム推移（kg・同部位の時間推移用）と、**部位別セット数**（月別積み上げ・部位間バランス用）。
   ※ kgは種目で水増しされるので、部位の横比較は**セット数**で見ること。
3. **継続** — 月別トレ回数（休んだ月は空白で残る）。
4. **体** — 体重を記録すると、体重推移と**相対筋力**（ベンチ推定1RM÷体重）が出る。
- ほか、ヒーロー（ベンチ推定1RM最高値）、月平均トレ回数、**自己ベスト一覧**。

## 記録の追記

### ツール内で（おすすめ・コード不要・スマホ対応）
1. 最上部の「**✏️ 今日のトレを記録する**」を押すと記録モードに入る（1行目に今日の日付が自動で入る）。
2. 種目とセットを書く。**`60x8.7.6` のように半角 `x` や `*` でもOK**（全角 `×` も可）。体重は `体重 75.5` の行を足すだけ。
3. 書きかけは**自動保存**されるので、筋トレ中に画面が消えても消えない。最後に「計算する」で確認 →「**記録する**」で**この端末（ブラウザ）に保存され、すぐグラフに反映**される。
4. 追記分は data.js の元データに上乗せ表示。下部「データ管理 / 書き出し」に「追記した記録: N件」と出る。

> 保存先はそのブラウザ内（localStorage）。**別の端末やブラウザには引き継がれない**点に注意。
> 永続バックアップや端末間で共有したいときは「data.js用に書き出す」でコードを出し、`data.js` に貼る（または Claude に「workout更新して」と頼む）。「全消去」で端末の追記分をリセットできる（data.js の元データは消えない）。

### Claude に頼む
メモを貼って「workout更新して」と言えば追記します。

### 手書き
```js
// data.js の WORKOUT_DATA に追加
{ date: "2026-06-20", items: [
  { name: "ベンチプレス", sets: "60×8.7.6" },
] },
// 体重は BODYWEIGHT に追加
{ date: "2026-06-20", kg: 72.5 },
```

## 記法ルール

| 記法 | 例 | 意味 |
|---|---|---|
| 新 `W×R1.R2.R3` | `30×10.10.10.10` | 30kgを10回×4セット |
| 新 `W×R1.R2` | `50×9.8` | 50kgを9回・8回（2セット） |
| 旧 `W×R×S` | `47×6×3` | 47kgを6回×3セット |
| 重量のみ | `46` | 重量だけ（ボリューム集計外・最大重量のみ反映） |

- ボリューム = Σ(重量 × 回数)。複数重量はスペース区切り。
- **自重・アシスト種目は対象外**（懸垂・腕立て・自重ディップス等。加重ディップス `+15×8` は対象）。

## 種目 → 部位（表記ゆれ自動統合）

| 部位 | 主な種目 |
|---|---|
| 胸 | ベンチプレス / インクライン / チェストプレス / フライ / ダンベルプレス / ディップス(加重) / デクライン |
| 背中 | ラットプルダウン / ローイング(各種ロー・tバー) / ナローグリップ / プルオーバー / 加重懸垂 |
| 脚 | レッグプレス / スクワット(ハック・ブラジリアン) / レッグエクステンション / カーフレイズ |
| 肩 | ショルダープレス / ミリタリープレス / アーノルド / サイドレイズ / リアレイズ |
| 腕 | アームカール / ダンベルカール / トライセプスエクステンション / スカルクラッシャー |

種目を増やすときは `index.html` の `classify()` に1行足す。
